What is mxAjax?

mxAjax is easy Ajax implementation framework for coldfusion developers. It is inspired from CFAjax but is build from grounds up with extensibility, maintainability and easy of usage in mind. CFAjax project was the initial attempt in providing Ajax framework to CF community, which is widely accepted by developers and used across many implementations. mxAjax not only provides the functionality that CFAjax did, but also removes deficiencies and incorporates suggestions provided by CFAjax users.
